## 📜 개요 (Outline)

**엘리스 SW 엔지니어 트랙 2기** 프로젝트 과제로서, 실제 서비스 운영이 아닌 기능 구현에 초점을 맞춘 데모 웹 어플리케이션입니다. 엘리스 측의 구현 요구사항에 따라, **Express.js**를 기반으로 **MPA(Multi Page Application)**를 구현하였습니다.
